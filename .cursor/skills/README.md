# Seed Garden Project Skills

This folder contains project-specific skills for building and iterating the Seed Garden website.

## Recommended Skills

- `figma-showcase-implementation`: Convert selected Figma frames into semantic, responsive web sections.
- `meditation-ui-polish`: Keep visuals calm, breathable, and aligned with meditation product tone.
- `mobile-first-responsive-qa`: Verify layout quality from 320px mobile through desktop.
- `mindful-copy-flow`: Write gentle product copy and section flow for emotional support journeys.

## How To Use

Ask the agent with clear intent, for example:

- "Use figma-showcase-implementation to add a new section from this frame."
- "Run mobile-first-responsive-qa on the current landing page."
- "Apply meditation-ui-polish to improve visual calm and spacing."
- "Use mindful-copy-flow to rewrite the hero and CTA."
